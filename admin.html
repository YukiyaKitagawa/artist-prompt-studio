Artist Prompt Studio Admin

絵師ペルソナ、キャラクターカテゴリ、キャラクター設定を管理します。 Private Storageへの画像アップロードにも対応しています。

管理ログイン

Supabase Edge FunctionのSecretに設定した ADMIN_PASSWORD を入力してください。

管理メニュー

絵師ペルソナ管理

index.htmlで選択できる絵師ペルソナを管理します。

キャラクターカテゴリ管理

キャラクターを分類するカテゴリを管理します。

キャラクター管理

キャラクター名、基本プロンプト、Private Storage内の画像パスを管理します。

画像アップロード

PNG / JPEG / WebP に対応しています。アップロード後、Storage内パスが image_url に自動入力されます。

Preview

先に画像をアップロードしてから、キャラクター保存を押してください。
